IBKR Connection Plan

Do not connect live orders first. Connect market data and paper trading first.

Stage 1 - Manual use

Use this dashboard as a visual decision system. Open IBKR separately and place no trades until you understand the replay.

Stage 2 - Paper trading

Install IBKR Trader Workstation or IB Gateway, enable API, then run a small Python bridge locally. GitHub Pages cannot talk directly to IBKR because IBKR runs on your computer.

Connection architecture

GitHub Pages dashboard
        ↓
Local Python bridge on your PC
        ↓
IBKR TWS / IB Gateway API
        ↓
Paper trading account first
        ↓
Live account only after testing
