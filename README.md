# Liquidity Sweep Replay Dashboard V2

A beginner-friendly, GitHub Pages-ready dashboard for learning liquidity sweep day trading using replay mode, simple trade tickets and a local browser journal.

## Important

This is a learning tool, not financial advice and not an automated trading system. V2 uses sample data only. Do not connect real money until you understand the model and have tested it.

## Files

- `index.html` - the dashboard app
- `data.json` - sample signal data structure
- `config.json` - beginner risk settings
- `dashboard-guide.pdf` - step-by-step guide

## How to use locally

1. Unzip the folder.
2. Double-click `index.html`.
3. Use Market Radar to select a market.
4. Use Replay Mode to understand the setup sequence.
5. Use Trade Ticket to review entry, stop, target and risk.
6. Save TAKE or PASS decisions to the Journal.

## GitHub Pages setup

1. Create a new repository in GitHub.
2. Upload all files from this folder.
3. Go to Settings > Pages.
4. Source: Deploy from branch.
5. Branch: main.
6. Folder: root.
7. Save and wait for GitHub to publish the site.

## V2 limitations

- No live data.
- No IBKR connection.
- No automatic order placement.
- Journal is saved in your browser local storage.

## Recommended learning rule

Paper trade first. If using real money, start with £100 and risk only £1 per trade.
